📘 Instrucciones para subir a Streamlit Cloud

1️⃣ Entra a: https://share.streamlit.io
2️⃣ Inicia sesión con tu cuenta de Google o GitHub.
3️⃣ Sube esta carpeta (nico-umsnh.zip) o súbela primero a un repositorio de GitHub.
4️⃣ Asegúrate de incluir:
    - nico_umsnh4.py (tu archivo principal)
    - requirements.txt
    - carpeta .streamlit/
    - carpeta Videos/
5️⃣ Haz clic en "Deploy" y espera unos segundos.

⚙️ Streamlit usará automáticamente los paquetes de requirements.txt.
🎬 Los videos deben estar en la carpeta "Videos".
🔑 La API Key de Google está dentro del código Python.
